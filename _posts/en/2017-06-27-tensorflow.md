During the presentation, we’ll walk through a sample problem that machine learning can solve in order to demonstrate the power and usefulness of TensorFlow, the tool developed by Google. The ultimate goal of the presentation is to change the image that machine learning has made in the eyes of developers.

We’re addressing two misconceptions. Machine learning is supposed to be difficult because it’s heavily focused on mathematics. Thanks to new tools like TensorFlow, we want to demonstrate that this is no longer necessarily the case. Machine learning is also considered too “magical” to be reliable. It’s partly for these two reasons that it struggles to be exploited in some companies.

Machine learning is no longer reserved for researchers, it must be accessible to developers and the rest of the world, hence the title of this presentation: the democratization of machine learning.
