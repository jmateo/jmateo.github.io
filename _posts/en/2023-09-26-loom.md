The Loom project brought us two preview features in JDK 19: virtual threads and structured concurrency.
These two elements receive some updates in JDK 20.
The third element developed in Loom has also been released: a new thread-local variable model called scoped values.
This presentation reviews these features, the state of patterns that will be available in concurrent and asynchronous programming once Loom is fully available, and what we can expect for JDK 21, the next LTS version to be released next September.
