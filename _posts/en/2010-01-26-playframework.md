Play! is a next-generation Java Web framework without compromises: it brings all the agility and productivity of frameworks based on dynamic languages that Java had never been able to benefit from. Unlike many Java frameworks that are merely attempts to hide overly complex technologies such as JSF or EJB, Play! tackles Web development problems with Java in a completely innovative way and definitively breaks away from the proven non-viable practices of the JEE world.

Created by Web developers for Web developers, Play! offers a very high productivity environment thanks to a hyper-optimized development cycle with no compilation phase and no deployment phase. All this combined with error reports of unprecedented perfection, even going so far as to propose a solution to the problem. By relying completely on the Java language, it allows you to take advantage of the richness of its ecosystem as well as its most modern development and optimization tools.
