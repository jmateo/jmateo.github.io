title: Alexis Moussine-Pouchkine 
---
Alexis Moussine-Pouchkine aime le bon vin et les architectures Java.

Chez Sun Microsystems, il agit en tant qu'ambassadeur du projet GlassFish, un serveur d'application Java EE nouvelle génération. Cela lui permet de rencontrer de nombreux développeurs Java et d'autres utilisateurs de technologies open source lors de réunions et de conférences.

Il participe à plusieurs communautés open source, traduit des ouvrages techniques sur Java et XML en français, et a néanmoins une vie en dehors de Java !
