title: Alexandre Delègue
---
Alexandre Delègue est un ingénieur d'études chez SERLI, il travaille principalement sur les technologies Java / JavaEE / Spring mais aussi sur les enjeux d'intégration continue, les métriques de qualité et même récemment sur un projet de centralisation de logs.

Ses expériences passées l'ont également amené à travailler sur des applications métier ainsi que sur un site public, particulièrement sur les enjeux de sécurité, de performance et de scalabilité.
