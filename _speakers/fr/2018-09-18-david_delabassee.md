Mon nom est David Delabassée, je suis dans l'espace Java depuis les débuts de cette technologie. Je travaille actuellement dans l'équipe Serverless Developer Relations d'Oracle où je passe la plupart de mon temps sur [Fn Project](https://github.com/fnproject).

Avant cela, je travaillais en tant que consultant chez Oracle dans l'organisation Java EE et j'ai participé récemment à l'initiative de transition du développement de la plateforme Java EE vers la Fondation Eclipse, alias [Jakarta EE](https://jakarta.ee/).

Je suis également un conférencier régulier aux [conférences et groupes d'utilisateurs, grands et petits, locaux et mondiaux](http://delabassee.com/conferences/).
