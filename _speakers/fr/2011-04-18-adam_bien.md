title: Adam Bien

Consultant et auteur, Adam Bien est membre du groupe d'experts pour les JSR Java EE 6, EJB 3.1 et JPA 2.0.

Il travaille avec la technologie Java depuis JDK 1.0 et Servlets/EJB 1.0 dans plusieurs projets à grande échelle et est actuellement architecte et développeur dans les projets Java SE/EE/FX.

Il a édité plusieurs livres sur Java et J2EE / Java EE et est l'auteur de [Real World Java EE Patterns](http://www.amazon.com/Real-World-Patterns-Rethinking-Practices/dp/0557078326).

Adam est un Java Champion, Oracle Java Developer de l'année 2010 et Rock Star JavaOne 2009.
