Mathieu ANCELIN est un développeur chez SERLI, spécialisé dans la programmation web et les frameworks web modernes avec une bonne connaissance des technologies Java EE.

Mathieu est impliqué dans plusieurs projets open-source comme GlassFish et Weld, et dirige également certains comme ReactiveCouchbase et Weld-OSGi ; Il est membre du groupe d'experts JSR-346 (CDI 1.1), un membre du groupe EEG OSGi et fait partie de l'équipe du Java User Group Poitou-Charentes.
