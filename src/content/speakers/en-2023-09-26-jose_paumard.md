José is a member of the Java Platform group at Oracle as a Java Developer Advocate.
He was a lecturer at Paris 13 University for 25 years and worked with companies on training and expertise.
He manages the technical blog "Java le soir" and contributes to various Open Source projects.
Involved in the Java community, he is treasurer of the BJPC association, organizer of Paris JUG evenings and co-founder of Devoxx France which he co-organized for the first 3 years.
Java Champion, regular speaker and multiple times Java Rockstar for Devoxx (FR, BE, UK), JavaOne, many French-speaking Java User Groups, he is also an author for Java Magazine.
