José est un membre du groupe Java Platform chez Oracle en tant que Java Developer Advocate.
Il a été maître de conférences à l'Université Paris 13 pendant 25 ans et a travaillé avec des entreprises sur la formation et l'expertise.
Il gère le blog technique « Java le soir » et contribue à divers projets Open source.
Impliqué dans la communauté Java, il est trésorier de l'association BJPC, organisateur des soirées Paris JUG et cofondateur de Devoxx France qu'il a co-organisé pendant les 3 premières années.
Java Champion, orateur régulier et multiple fois Java Rockstar pour Devoxx (FR, BE, UK), JavaOne, de nombreux Java User Groups francophones, il est également auteur pour Java Magazine.
