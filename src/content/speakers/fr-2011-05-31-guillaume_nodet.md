title: Guillaume Nodet
locale: fr
---
Guillaume Nodet est un Software Fellow chez FuseSource, une filiale de Progress Software offrant des produits SOA (Service Oriented Architecture) basés sur les principaux projets de la Fondation Apache.

Guillaume a une expérience significative dans la communauté open source et est un contributeur très respecté sur plusieurs projets, notamment Apache ServiceMix, Apache ActiveMQ, Apache Camel.

Il est membre de la Fondation Apache et vice-président pour Apache Karaf. Guillaume est également impliqué dans les groupes d'experts de l'Alliance OSGi.
