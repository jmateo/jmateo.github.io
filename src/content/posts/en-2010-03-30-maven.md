Arnaud Héritier, member of the Apache Maven and Codehaus Mojo communities and author of the Apache Maven book, comes to talk to us about Maven usage in enterprise and the new features of version 3.x.
Software Factory Manager for eXo Platform, Arnaud will share his experience using Maven in a professional software forge: what Maven brings to each stage of software construction.
We will address how Maven is involved in each of these stages:
 * project build on the developer’s workstation
 * dependency and repository management
 * setting up continuous integration
 * use of metrics and quality reports
 * continuous deployment
