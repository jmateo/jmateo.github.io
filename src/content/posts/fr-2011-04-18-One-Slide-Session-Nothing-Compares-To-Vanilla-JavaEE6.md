Java Champion Adam Bien, auteur de Real World Java EE Patterns, va jeter son diaporama par-dessus bord et démontrer, à partir de zéro, combien il est facile de créer une application Java EE 6 simple, légère et facile à développer.

Vous ne croyez pas au battage publicitaire autour de Java EE 6 étant léger et facile à utiliser ? Vous n'avez toujours pas pardonné à l'entreprise Java les erreurs de J2EE ? Si vous avez quitté la norme d'entreprise Java pour un autre cadre de travail, prenez une heure de votre journée pour assister à cette démonstration interactive, et sans aucun doute, Java EE 6 vous reconquerra.

Regardez et participez alors qu'Adam développe une application Java EE 6 avec une interface utilisateur JSF 2.0 exécutée sur EJB 3.1, utilisant REST, le traitement asynchrone, CDI et JPA 2 ; et tout cela avec des tests unitaires inclus, le tout en une heure. Et c'est tout fait à partir de zéro, sans assistants, templates ou génération de code. Cela est rarement possible avec une autre plateforme, mais c'est « du travail ordinaire » en Java EE 6.

Cette session démontre les nouvelles fonctionnalités de Java EE 6 dans un style de démonstration continue. Les questions des participants recevront des réponses avec du code fonctionnel
