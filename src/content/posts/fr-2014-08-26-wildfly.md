WildFly 8 (anciennement JBoss Application Server) est le serveur d'applications open source conforme à Java EE 7 de Red Hat.
Il contient des implémentations robustes de WebSocket, Batch, JSON, Concurrency, JMS2, JAX-RS 2, CDI 1.1 et toutes les technologies Java EE 7.
Undertow est le nouveau serveur web de pointe dans WildFly 8 et est conçu pour un débit et une scalabilité maximums, y compris les environnements avec plus d'un million de connexions. Le nombre de ports utilisés est réduit en multiplexant les protocoles sur HTTP en utilisant HTTP Upgrade.
Cette présentation fournira un aperçu de toutes ces fonctionnalités et bien d'autres, telles que le mode domaine, RBAC, etc.
