Spring Boot 2 is the first major release since the first version of the project in April 2014. In addition to leveraging Spring Framework 5 and the new versions of Spring projects, it brings many features and even new programming models.

It's also a unique opportunity to fix and deeply improve some of its mechanisms.

In this session, Stéphane will upgrade a Spring Boot 1.5 application to 2.0 and will show by example how Spring Boot 2 improves the developer experience, including during the migration.
