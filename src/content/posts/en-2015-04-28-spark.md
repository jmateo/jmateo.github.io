Today, Google, Facebook and Twitter do business with data! We have more and more data at our disposal, which represents real stakes and real value. We therefore need tools to store, process and analyze it.

Let’s discover two tools that have been making a lot of noise recently!

Apache Cassandra is a scalable NoSQL database, multi-datacenter and very simple to deploy, which allows storage of large quantities of data and is suitable for different data models.

Apache Spark is a distributed and in-memory data processing framework, which perfectly fits the analysis of large data volumes.

During this presentation, we will define the key concepts of Cassandra and Spark, then those of the Spark-Cassandra connector and the Spark ecosystem (MLlib, Spark streaming…).

We will conclude with a demo based on data from an accelerometer, during which we will see how to process and analyze it using Cassandra and its connector, Spark and its Machine Learning libraries (MLlib).
