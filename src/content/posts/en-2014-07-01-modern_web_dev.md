The Web environment has evolved significantly in recent years and now offers a level of industrialization comparable to the Backend. Web browsing now occurs across a multitude of devices and the trend is towards standardizing sites across all types of platforms. Based on the latest HTML5 and CSS3 features, frameworks like Bootstrap and Foundation are powerful tools for creating modern web sites adapted to all devices.

This presentation introduces the concepts of Responsive Web Design, then reviews the different Web languages (HTML, CSS, JavaScript). Subsequently, it reveals the possibilities offered by the most popular JavaScript frameworks (AngularJS, Backbone.js) and highlights the tools that Web Developers now have at their disposal to create, manage and automate the multiple tasks necessary for web site creation.
