Java 8 est la mise à jour la plus impactante de l'écosystème Java depuis Java 5, introduisant un modèle de programmation entièrement nouveau avec les expressions lambda, et une toute nouvelle plateforme d'applications avec support des appareils ARM intégrés.
Dans cette présentation, vous apprendrez comment tirer parti des nouvelles API Java 8 activées par les lambdas, y compris les collections améliorées, les interfaces fonctionnelles et les gestionnaires d'événements simplifiés.
De plus, vous verrez des démonstrations en direct de la technologie Java 8 exécutée sur des plates-formes intégrées telles que Raspberry Pi et Lego Mindstorms EV3, permettant aux applications basées sur Java de soutenir l'internet des objets.
